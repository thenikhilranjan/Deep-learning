# WiSARD dataset (local only)

Place the **WiSARDv1** tree here so the layout matches what the code expects, for example:

```text
datasets/WiSARDv1/<sequence_folders>/...
```

The full corpus is **very large** (tens of GB of imagery). **GitHub cannot host it** inside this repository (file and repo size limits), so the actual image and label files are **not committed**.

- Keep your copy on disk under this folder for training and the Streamlit app.
- For submission or collaboration, share the dataset via your institution’s approved channel (USB, cloud storage, LMS) alongside this code repository.

See `docs/setup/DATASET_READY.md` in the project root for environment notes.
